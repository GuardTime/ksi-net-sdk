HAAggregatorConfigRequestRunner
Class that is resposible for running high availablity sub-service aggregator configuration requests and giving consolidated successful sub-service results as a result.
HAAsyncResult
High availablity KSI service async result.
HAExtenderConfigRequestRunner
Class that is resposible for running high availablity sub-service extender configuration requests and giving consolidated successful sub-service results as a result.
HAExtendRequestRunner
Class that is resposible for running high availablity sub-service extending requests and giving the first successful sub-service result as a result.
HAKsiService
High availability KSI service. Combines max 3 sub-services to achieve redundancy.
HARequestRunner
Class that is resposible for running high availablity sub-service requests and giving a result based on the requests. In case signing request we need to wait HA service end request call (EndSign or GetSignResponsePayload) before sub-services end request calls can be made.
HASignRequestRunner
Class that is resposible for running high availablity sub-service signing requests and giving the first successful sub-service result as a result.
